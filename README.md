# Real World Software Development – Java Projects

# English

This repository contains projects developed based on the concepts presented in the book "Real-World Software Development: A Project-Driven Guide to Fundamentals in Java", published by O’Reilly and written by Raoul-Gabriel Urma and Richard Warburton.

The projects are educational in nature, focusing on the application of software engineering best practices, clean design, automated testing, and incremental code evolution using Java.

---

## Project Objective

The main goals of this repository are to:

- Apply real-world software development concepts  
- Evolve simple solutions into more robust architectures  
- Practice object-oriented and functional design in Java  
- Apply automated testing as part of the development process  
- Demonstrate clean code organization and professional best practices  

This repository can also be used as a technical portfolio for academic and professional purposes.

---

## Developed Projects

The projects follow the incremental approach proposed in the book. Among them:

- **Bank Statement Analyzer**  
  A Java application that processes bank statement files (CSV), calculates statistics, and evolves throughout the project to incorporate:

  - Improved separation of responsibilities  
  - Use of interfaces and abstractions  
  - Unit testing  
  - More flexible and maintainable code

# Português

Este repositório contém projetos desenvolvidos com base nos conceitos apresentados no livro "Desenvolvimento Real de Software: Um guia de projetos para fundamentos em Java", publicado pela O’Reilly e escrito por Raoul-Gabriel Urma e Richard Warburton.

Os projetos têm finalidade educacional, com foco em aplicar boas práticas de engenharia de software, design limpo, testes automatizados e evolução incremental de código em Java.

---

## Objetivo do Projeto

O objetivo principal deste repositório é:

* Aplicar conceitos de desenvolvimento de software no mundo real
* Evoluir soluções simples para arquiteturas mais robustas
* Praticar design orientado a objetos e funcional em Java
* Aplicar testes automatizados como parte do processo de desenvolvimento
* Demonstrar organização de código e boas práticas profissionais

Este repositório também pode ser utilizado como portfólio técnico para fins acadêmicos e profissionais.

---

## Projetos Desenvolvidos

Os projetos seguem a abordagem incremental proposta no livro. Entre eles:

* **Analisador de Extratos Bancários**
  Aplicação Java que processa arquivos de extrato bancário (CSV), calcula estatísticas e evolui ao longo do projeto para incorporar:

  * Melhor separação de responsabilidades
  * Interfaces e abstrações
  * Testes unitários
  * Código mais flexível e manutenível
